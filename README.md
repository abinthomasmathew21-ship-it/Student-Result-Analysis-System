# 🎓 Student Result Analysis System

## 📌 Project Overview

The **Student Result Analysis System** is a Python-based Data Analytics project developed to manage, analyze, and visualize student academic performance.

This project processes student records from CSV files, stores data using SQL databases, performs grade calculations, identifies toppers, analyzes attendance, generates reports, and provides meaningful insights through data analysis techniques.

It is designed as an educational project to demonstrate practical applications of:

* Python Programming
* SQL Database Management
* CSV Data Processing
* Data Analysis
* Data Visualization
* Academic Performance Reporting

---

# 🚀 Key Features

### 📊 Student Result Analysis

* Store and process student academic records
* Calculate total marks and percentage
* Generate student performance reports

### 🏆 Topper Identification

* Identify highest-performing students
* Display Top 3 students based on percentage

### 🎯 Grade Calculation

* Automatic grade assignment

| Percentage | Grade |
| ---------- | ----- |
| 90+        | A+    |
| 80-89      | A     |
| 70-79      | B+    |
| 60-69      | B     |
| Below 60   | C     |

### ✅ Pass / Fail Analysis

* Determine student result status
* Generate pass percentage statistics

### 📚 Subject-wise Analysis

* Highest mark in each subject
* Lowest mark in each subject
* Average mark in each subject

### 🔍 Student Search System

* Search student records by name
* Quickly retrieve academic information

### 🗄 SQL Database Integration

* Store student records using SQLite
* Perform structured data management

### 📈 Data Visualization

* Generate performance charts
* Analyze trends using graphical representation

### 📅 Attendance Analysis

* Attendance tracking
* Low attendance identification
* Average attendance calculation

# ⚙️ How to Run the Project

## Step 1

Clone Repository

```bash
git clone https://github.com/abinthomasmathew21-ship-it/Student-Result-Analysis-System.git
```

## Step 2

Install Dependencies

```bash
pip install -r requirements.txt
```

## Step 3

Run Python Files

```bash
python main.py
```

OR

```bash
python analysis.py
```
